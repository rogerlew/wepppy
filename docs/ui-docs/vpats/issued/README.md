# Issued VPAT Archive

This directory is append-only history for buyer-facing VPAT / ACR issues that were actually cut.

## Rules

- One directory per issued package, not per commit.
- Directory name format: `YYYY-MM-DD_<shortsha>/`
- Do not edit an issued package to reflect later product changes.
- If a clerical correction is unavoidable, add a correction note and preserve the original issue context.
- If a later issue supersedes an earlier one, record that in the newer package README.

## Expected Contents Of Each Issued Package

- final buyer-facing VPAT / ACR artifact
- README with scope and supersession notes
- evidence index used for that issue
- frozen environment matrix
- frozen source snapshot metadata
