# Current VPAT Scope

This file freezes the intended scope for the next buyer-facing VPAT / ACR issue.

## In Scope

- WEPPcloud web application
- anonymous landing and interfaces pages
- authenticated run-management workflows
- report templates and representative report-accessibility probe surfaces
- public support and accessibility statement pages served within the WEPPcloud application

## Out Of Scope

- third-party sites linked from WEPPcloud
- external video platforms
- hardware
- non-product institutional websites
- non-web document formats not yet explicitly added to the evidence boundary

## Scope Change Triggers

Refresh this file and the worksheet when:

- a new user-facing surface becomes part of the federal-buyer deployment
- an existing in-scope surface is removed or materially redesigned
- support documentation scope changes
- non-web documents become part of the buyer-facing package
