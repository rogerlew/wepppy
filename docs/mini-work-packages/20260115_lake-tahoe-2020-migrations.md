# Mini Work Package: Lake Tahoe 2020 Watershed Migrations
Status: Pending
Last Updated: 2026-01-15
Primary Areas: Lake Tahoe Basin watersheds (`lt_202012_*`)

## Objective
Run migrations on all 799 Lake Tahoe 2020 watershed runs across 11 scenarios.

## Scope
- 80 base watersheds (including intervening areas and sub-watersheds)
- 11 scenario types per watershed

## Scenarios
| Scenario | Description |
|----------|-------------|
| `CurCond` | Current conditions baseline |
| `PrescFire` | Prescribed burning |
| `LowSev` | Low severity fire |
| `ModSev` | Moderate severity fire |
| `HighSev` | High severity fire |
| `Thinn85` | Skidder thinning (85% cover) |
| `Thinn93` | Cable thinning (93% cover) |
| `Thinn96` | Thinning (96% cover) |
| `SimFire.fccsFuels_obs_cli` | FCCS fuels, observed climate |
| `SimFire.landisFuels_obs_cli` | LANDIS fuels, observed climate |
| `SimFire.landisFuels_fut_cli_A2` | LANDIS fuels, future climate A2 |
