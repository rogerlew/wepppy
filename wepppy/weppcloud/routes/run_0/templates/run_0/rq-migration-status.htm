What are migrations?

Migrations are automated updates that convert older data formats to newer, more efficient formats. They improve load times, reduce file sizes, and ensure compatibility with the latest features.

All migrations are idempotent, meaning they can be safely run multiple times without causing problems. If a migration has already been applied, it will be skipped.
