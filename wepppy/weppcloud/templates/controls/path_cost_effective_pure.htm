Omni Preconditions

PATH consumes Omni artifacts you provision before running it: Omni scenarios for the post-fire baseline (sbs_map), undisturbed, and each configured mulch treatment, plus Omni contrasts (treatment vs post-fire control) — the outlet sediment-discharge constraint is built from contrast deltas, so contrasts are required. Missing artifacts fail the run with a message naming what to provision.

Note: on runs with grouped contrasts, the burn-severity filter uses each group's dominant landuse class and can be unmapped when unburned hillslopes dominate a group — prefer it on per-hillslope runs.

Thresholds
tonne/yr

Maximum allowed average-annual sediment discharge at the outlet after treatment. Set at or above the post-fire value to relax the outlet constraint.

Per-hillslope yield ceiling; hillslopes above it must be treated. Units follow PATH's prepared data (tonnes per acre).

Filters

Hold Ctrl/Command to select multiple severities. Leave empty to include all.

Treatments
Label Omni Scenario Rate (tons/acre) Unit Cost ($/acre) Fixed Cost ($)

Unit costs are stored in USD per acre (displayed per hectare when SI units are selected); per-hillslope cost = area × unit cost × rate. The label and mulch rate derive from the selected Omni scenario (mulch_15_sbs_map → 0.5 tons/acre). Each treatment's Omni scenario and contrasts must exist before running.
