WEPPcloud Return Period Analysis


The Return period of the events are estimated by applying Weibull formula on annual maxima series:

T = (N + 1)/m,
where T is the return period, N is the number of simulation years, and m is the rank of he annual maxima event.

For post-disturbance model runs (e.g. prescribed fire, wildfire, thinning, etc.) results represent recurrences for specific parameters given the first year post-disturbance only. For example, if the peak discharge for a 20 years return interval is 5 m3/s, this means that if we were to apply a specific treatment every year for 20 years in a row, we would observe the 5 m3/s peak discharge only once.
