# Geneva Work-Package Evidence

This directory stores execution evidence for Geneva work-packages listed in:

- `/workdir/wepppy/wepppy/nodb/mods/geneva/implementation-plan.md`

Usage:

- Create one evidence file per work-package:
  - `wp-00_*.md`, `wp-01_*.md`, ...
- Keep evidence append-only per work session.
- Record commands, test results, review findings, and manual integration notes.
- Link relevant commits and PRs.

Required minimum sections for each evidence file:

1. Scope Implemented
2. Code Changes
3. Automated Tests Run
4. QA Review
5. Security Review
6. Manual Integration Checks
7. Findings and Disposition
8. Exit-Criteria Check
